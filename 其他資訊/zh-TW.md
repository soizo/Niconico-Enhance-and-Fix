# Niconico 增強與修復

統計異常或日區帳號自動切換到嵌入播放器；原播放器嘗試跳過/屏蔽廣告；保留嵌入播放器主要控制。

[adblock CSS](https://greasyfork.org/scripts/561612-nicovideo-adblock)

## 功能
- 當播放數/留言數等統計顯示不合理時，自動切換到嵌入播放器。
- 偵測到日語/日本相關帳號/語言環境時，自動切換到嵌入播放器。
- 在原（非嵌入）播放器中盡量跳過/快進廣告，並嘗試點擊「跳過/スキップ/skip」等按鈕。
- 透過 CSS 隱藏觀看頁常見廣告區域。
- 在嵌入播放器頁面中，透過 CSS 隱藏部分廣告/遮罩 UI。
- 嵌入模式下保留關鍵控制能力：
  - 將觀看頁的「設定」按鈕移到靠近全螢幕按鈕的浮層控制區，確保可點可用。
  - 維持浮層控制的可點擊性與顯示/隱藏聯動。
- 倍速同步與解鎖：
  - 主頁倍速變更同步到嵌入播放器。
  - 嵌入播放器按設定倍速持續套用。
  - 嘗試解鎖/補齊更高倍速選項（x1.5 / x1.75 / x2.0），並移除停用狀態。

## 備註
- 匹配範圍：`www.nicovideo.jp/watch/*`、`nicovideo.jp/watch/*`、`embed.nicovideo.jp/watch/*`
- 透過 DOM 監聽與定時掃描維持嵌入、按鈕位置與倍速同步。
