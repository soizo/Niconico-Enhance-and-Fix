# ニコニコ強化＆修正

再生数/コメント数などの表示が不自然なとき（または日本向けアカウント）に埋め込みプレイヤーへ自動切替。広告は可能な範囲でスキップ/ブロックし、埋め込み時も主要操作を使えるようにします。

## 機能
- 再生数・コメント数の表示が不自然な場合に埋め込みプレイヤーへ自動切替。
- 日本向けロケール（日本アカウント想定）の場合は埋め込みプレイヤーへ自動切替。
- 元の（非埋め込み）プレイヤーでは可能な範囲で広告をスキップ/早送りし、「スキップ」ボタンがあればクリック。
- 視聴ページ上の一般的な広告枠を CSS で非表示。
- 埋め込みプレイヤー側の一部広告/オーバーレイ UI を CSS で非表示。
- 埋め込み時も主要操作を維持:
  - 視聴ページの「設定」ボタンをフルスクリーン付近のオーバーレイ領域へ移動して操作可能に。
  - オーバーレイ操作をクリック可能に維持。
- 再生速度の同期:
  - 視聴ページ側で変更した再生速度を埋め込みへ送信して反映。
  - 埋め込み側でも指定速度を維持/適用。
  - x1.5 / x1.75 / x2.0 を可能な範囲で解除・追加（無効項目を有効化）。

## 補足
- 対象: `www.nicovideo.jp/watch/*`、`nicovideo.jp/watch/*`、`embed.nicovideo.jp/watch/*`
- DOM 監視と定期スキャンで埋め込み・ボタン位置・再生速度を追従します。
