# Niconico 增强与修复

统计异常或日区账号自动切换到嵌入播放器；原播放器尽量跳过/屏蔽广告；保留嵌入播放器主要控制按钮可用。

[adblock CSS](https://greasyfork.org/scripts/561612-nicovideo-adblock)

## 功能
- 当播放量/评论数等统计显示不合理时，自动切换到嵌入播放器。
- 检测到日语/日本相关账号/语言环境时，自动切换到嵌入播放器。
- 在原（非嵌入）播放器中尽量跳过/快进广告，并尝试点击“跳过/スキップ/skip”等按钮。
- 通过 CSS 隐藏观看页常见广告区域。
- 在嵌入播放器页面中，通过 CSS 隐藏部分广告/遮罩类 UI。
- 嵌入模式下保留关键控制能力：
  - 将观看页的“设置”按钮移动到靠近全屏按钮的浮层控制区，保证可点可用。
  - 维持浮层控件的可点击性与显示/隐藏联动。
- 倍速同步与解锁：
  - 主页面倍速变化同步到嵌入播放器。
  - 嵌入播放器按设定倍速持续应用。
  - 尝试解锁/补充更高倍速选项（x1.5 / x1.75 / x2.0），并移除禁用状态。

## 备注
- 匹配范围：`www.nicovideo.jp/watch/*`、`nicovideo.jp/watch/*`、`embed.nicovideo.jp/watch/*`
- 通过 DOM 监听与定时扫描保持嵌入、控件位置与倍速同步。
